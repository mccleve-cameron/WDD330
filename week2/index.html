Comments

Single line comments start with // and finish at the end of the line:
// this is a short comment
Multiline comments start with /* and finish with */:
/* This is a longer comment anything here will be ignored This is a useful place to put notes *

Syntax

Semicolons are not required because of Automatic Semicolon Insertion (ASI). It will try and place semicolons at the end of lines. However it can cause errors, so its porbably best to be safe and add them.

Dot Notation
name.length; -returns number of characters in a string

Bracket Notation
name['length']; -also returns the number of characters.

Converting Strings to Numbers

To covert a string into a number we can multiply a numerical string by 1, which will convert it into a number because of type coercion: answer = "5" * 1; = 5
Yet another way to convert a string into a number is to use the Number function: Number("23"); = 23

Arrays

arrays can be declared by typing just square brackets '[]'
Items can be added to an array by doing; pizza[0] = pepperoni
Items can be removed by doing; delete pizza[0];
.shift and .unshift add and remove items from the bottom/front of the array
join() method takes an array and makes them into a long string.

Functions

Functions are delcared using the key word function, then variable name, then parenthesis.
Functions are first-class objects, they behave the same as any value. A function passed as and argument is called a callback.
