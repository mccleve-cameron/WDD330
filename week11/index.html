Week 11

Check out my Block 2 Project:

Bodybuilding Bible

Description:

Bodybuilding Bible is a web application that gives you access to tons of exercises targeting any muscle group you choose. When you first log in you will see a 2-dimensional human muscle anatomy. Then you can then click on any muscle and it will take you to a page that will list exercises to target that muscle group.

The application uses Node.js to communicate with a database I created on heroku. The exercises are loaded dynamically from the database based on the user's selection. Currently the images only load when using the Chrome browser.
