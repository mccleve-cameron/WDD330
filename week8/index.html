Transforms

  • They were finally introduced in CSS3 and are now available on all modern browsers.
  • It allows you to translate, rotate, scale, and skew any element.

  • Traslate

    • Move elements left, right, up, or down
    • Code examples: transform: translate(45px, -45px);
    • Click Me!

  • Rotate

    • Rotate and element around the point of origin
    • Code example: transform: rotate(10deg) translateX(40px) scale(1.5);
    • Click Me!

  • Scale

    • Grow and shink and object
    • Code example: transform: scale(1.5, 0.25);
    • Click Me!

  • Skew

    • Skews an element along the x and y axes
    • Code example: transform: skew(15deg, 4deg);
    • Click Me!
